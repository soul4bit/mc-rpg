# Redstone Launcher Java Assets

Набор PNG-ассетов, вытащенных из утверждённого минималистичного дизайна лаунчера Redstone.

## Структура

- `mockups/` — полные экраны для ориентира при верстке: home, play, mods, settings, account, login, register.
- `background/` — затемнённые/размытые фоновые изображения 1920x1080 для JavaFX/Swing.
- `logo/` — логотип Redstone и кристалл. Версия `*_transparent.png` имеет примерную прозрачность, так как исходник был вырезан из тёмного макета.
- `avatars/` — аватар игрока.
- `thumbnails/` — миниатюры новостей и сервера.
- `mod_icons/` — иконки модов для окна Mods.
- `ui_icons/` — вырезанные иконки меню/утилит и несколько UI-сниппетов.

## Для JavaFX

Фон можно ставить через `ImageView` на весь корневой контейнер, а панели делать обычными `Pane/VBox/HBox` с CSS: `rgba(12,14,18,0.78)`, `border-radius: 18px`, красный акцент `#e82929`.

Примечание: кнопки, карточки, input-поля, прогресс-бары и тумблеры лучше не делать картинками, а верстать нативными JavaFX/Swing компонентами. Изображения здесь нужны в основном для логотипа, фона, аватаров, миниатюр и иконок.
